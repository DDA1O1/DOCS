### Setting up the runtime environment

- Download runtime environment such as [Node.js](https://nodejs.org/), [Bun](https://bun.sh/), [Deno](https://deno.com/).Traditionally,Javascript was used only on the client side(in the browser). Node.js is the most popular runtime environment with vast ecosystem of libraries(npm).Built on the [V8 JavaScript engine](https://github.com/v8/v8) (the same engine used by Chrome), Node.js enables developers to use JavaScript to build server applications, command-line tools, APIs, and more.
