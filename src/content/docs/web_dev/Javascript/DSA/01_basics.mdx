## DSA

- It's all about how to store, organize data and which algorithm to use so to achieve a desired outcome.

```ascii
           └──────────┬──────────────────┘
                      │
                      ▼
                ┌──────────────┐
                │   OUTPUT     │
                │  (Solution)  │
                └──────────────┘
```
